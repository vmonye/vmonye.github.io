如何用 hugo 搭建个人博客

安装 Hugo 到Hugo releases页面 下载hugo_xxx_Windows-64bit.zip 解压文件，将hugo.exe放到D:\Software\hugo\hugo.exe 将D:\Software\hugo\加到PATH 重启终端，运行hugo version 来查看版本 快速搭建博客 进入hugo官网 点击 Quick Start 操作从step2开始里面的代码直到step7，得到public目录，public目录就是一个网站 在目录外新增.gitignore文件并输入/public/，让这个仓库不要存储它 运行git init ， git add . ， git commit 命令，进行部署 打开GitHub新建一个repo 名称必须为：用户名.github.io，并把public中的内容上传 进入Settings中的GitHub Pages下可以看到http://你的用户名.http://github.io的链接
